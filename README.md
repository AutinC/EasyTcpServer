# EasyTcpServer
在Linux环境下使用C++语言开发多线程服务器通信框架，主要实现百万级数据包的稳定收发，支持一定数量的客户端同时连接并及时提供响应。允许依托该框架扩展相应的业务逻辑代码实现不同功能。
# 主要工作：
1、使用C++和socket编程，利用SELECT、EPOLL模型实现I/O多路复用；

2、采用包头与包体的格式来进行数据包的收发，解决了TCP字节流传输造成的数据粘包问题；

3、服务端使用生产者-消费者模式设计，分离接收和网络消息处理业务，建立消息缓冲队列作为代理在用户层面实现负载均衡，使用计时器实现接收和发送数量的计数；

4、添加内存管理模块，构建内存池、对象池减少内存波动，使用智能指针避免内存泄漏，使用C++11新特性中的function与lambda表达式封装收发消息任务类，减少了内存的申请与释放；

5、采用线程池技术配合互斥锁、信号量等技术来实现多线程同步，提高服务器的并发量；

6、完善服务端功能：心跳检测、日志打印、消息定时发送、定量发送、异步发送；

7、使用socket相应选项解决TIME_WAIT状态导致的连接问题、惊群现象，在内核层面实现负载均衡。
